# SIEM Policy IP Mapper
데이터센터 이전 시 SIEM 정책 내 장비 IP를 
자동으로 변환해주는 도구입니다.

## 왜 만들었나요?

데이터센터 이전 작업으로 인해 SIEM에 등록된
200건 이상의 정책 내 장비 IP를 전부 변경해야 했습니다.

수동으로 진행할 경우 직접 찾고 비교하며 수정해야 해서
최소 5시간 이상 소요될 것으로 예상했고,
휴먼에러 발생 가능성도 높았습니다.

구장비 IP와 신장비 IP를 1:1 매칭하여
자동으로 변환하는 도구를 직접 만들어 해결했습니다.

## 결과

- 소요시간: 5시간 → 1시간 이내로 단축
- 휴먼에러 제거: 자동 매칭으로 오탐 방지

## 동작 방식

1. SIEM에서 내보낸 정책 CSV 파일을 읽어들임
2. 구장비 IP / 신장비 IP를 쉼표로 구분해 입력
3. 장비 수가 일치하지 않으면 오류 처리 후 종료
4. 정책 내 IP가 포함된 컬럼을 순회하며 자동 변환
5. 변경된 항목에 변경여부 플래그(Y) 표기
6. 결과를 엑셀 파일로 출력 (원본 룰셋 / 변경 룰셋 포함)

## 기술 스택

- Python 3.x
- pandas: CSV 데이터 처리 및 IP 매칭 로직
- openpyxl: 엑셀 출력

## 실행 방법

# 패키지 설치
pip install pandas openpyxl

# 실행
python ip_mapper.py

- 실행 후 구장비 IP, 신장비 IP를 쉼표로 구분해 입력
- ex) 구장비: 192.168.1.1, 192.168.1.2
-     신장비: 10.0.0.1, 10.0.0.2
- 결과: 일반_result.xlsx 생성
